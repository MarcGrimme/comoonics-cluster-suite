Base tools written in python used for open sharedroot clusters with comoonics software.

Those are classes used by more other packages.

Functionality provided here are
comoonics.stabilized:      library for stabilizing files
comoonics.lockfile:        library for locking files
comooncis.odict:           library for a ordered dictionary.
comoonics.XMLConfigParser: library for config files based on XML.
comoonics.AutoDelegator:   Implementation of the auto delegator design pattern.
